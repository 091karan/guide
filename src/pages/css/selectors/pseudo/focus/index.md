## Focus

### Definition

The `:focus` represents an element that has received a focus state, triggered by an event from the keyboard. Trigger action comes from pressing the **TAB** key, which visually create a ring around an element.

### Syntax

`:focus`

### Example

```
a:focus {
  color: red;  
}
```
